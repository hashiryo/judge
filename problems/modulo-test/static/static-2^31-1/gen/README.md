# static-2^31-1

固定法 2^31-1=`2147483647` に対して、反復

`s <- (a * s + b) * (c * s + d) (mod 2147483647)`

を `N` 回行った最終値を出力するベンチ用の自作問題。

## 入力

1 行に 6 整数を与える。

`N s a b c d`

- `0 <= N`
- `0 <= s, a, b, c, d < 2147483647`

## 出力

`N` 回反復した後の `s` を出力する。

## 再生成

以下を問題ディレクトリで実行する。

```bash
./gen/generate.sh
```
