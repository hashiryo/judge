# runtime-31

入力で法 `mod` を与える、64 bit 動的 mod 用のベンチ問題。

反復は

`s <- (a * s + b) * (c * s + d) (mod mod)`

で、`mod` は毎ケースごとに与えられる。

## 入力

1 行に 7 整数を与える。

`N mod s a b c d`

- `0 <= N`
- `2^32 < mod < 2^40`
- `mod` は奇数
- `0 <= s, a, b, c, d < mod`

## 出力

`N` 回反復した後の `s` を出力する。

## ケース方針

- 小さい手書きケース
- 境界寄りの `mod`
- `2^32` 付近の最小側
- `2^40` 付近の最大側
- ランダムな奇数 mod
