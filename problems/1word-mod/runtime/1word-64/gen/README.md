# 1word-64

固定の `b` (`2^63 <= b < 2^64`) に対し、`a % b` を `N` 個の `a` について計算するベンチ問題。`a` は最上位ビットが立った 64 bit 整数 (`2^63 <= a < 2^64`)。

各 `a_i` は base.cpp 内で `(sa, ma, mb)` から決定的に生成する:

```
sa <- sa * ma + mb  (u64 wrap)
a  <- sa | (1 << 63)
acc ^= div.mod(a)
```

`b` は入力で 1 ケース 1 回与えられ、`struct DIV` のコンストラクタに渡される。前計算は計測対象外、`mod` ループのみを計測する。

## 入力

1 行に 5 整数を与える。

`N b sa ma mb`

- `0 <= N`
- `2^63 <= b < 2^64`
- `0 <= sa, ma, mb < 2^64`

## 出力

`acc` (各 `a_i` に対する `b` での剰余の累積 XOR) を出力する。

## ケース方針

- 小さい手書きケース
- 境界寄りの `b` (最小・最大など)
- 様々な N でのランダムケース
